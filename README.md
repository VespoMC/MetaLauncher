# MetaLauncher
## Launch Minecraft. Faster.

MetaLauncher is a minecraft launcher created for preformace. custom ui and backgrounds, instance managers, built in download from modrinth support, fully customizable,

The launcher supports microsoft accounts as well as temporarily supports offline/cracked accounts.

# IMPORTANT
this client is still in development. this is just a public pre-release. changes to the launcher as well as our own custom in-game client will be released in the full public update.
